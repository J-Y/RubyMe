Blog Me, you know, a platform for sharing blog.
Site url: ****************

## How to Install

Make sure mysql already started.
```bash
git clone git@github.com:jasonZYJ/blog_me.git
cd blog_me
# A ruby script to RTF
ruby setup.rb
rails s
```

## How to Test

We use rspec to test.
```bash
bundle exec rake rspec
```
